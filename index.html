This is a simple tool to make localizing Don't Move a bit easier.
Use "Edit" to change the text.
Set the style with the red buttons.
Download a PNG of the text with "Get".
Long text may appear clipped; the entire text will be exported.
Click and move the mouse to move the text around and preview how it will look.
Use left/right arrow keys to change letter thickness.
